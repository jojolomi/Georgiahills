# KPI Dashboard Access and Setup

## GA4
1. Property: Georgia Hills (production).
2. Create custom dimensions:
- `gcc_country`
- `lead_channel`
- `booking_variant`
3. Ensure events are tracked:
- `whatsapp_click`
- `call_click`
- `booking_modal_open`
- `booking_modal_whatsapp_submit`
- `booking_created`

## Google Search Console
1. Add and verify:
- `https://georgiahills.com`
- `https://georgiahills-15d19.web.app`
2. Submit sitemap:
- `https://georgiahills.com/sitemap.xml`
3. Create query filters for: sa, ae, qa, kw, bh, om audiences.

## Bing Webmaster Tools
1. Verify domain using `msvalidate.01` meta token.
2. Submit sitemap and monitor crawl/index stats weekly.

## Weekly Dashboard Views
1. Organic sessions by country.
2. Top queries by locale (`en`/`ar`).
3. CWV/Lighthouse trend on priority routes.
4. Leads from WhatsApp/call/modal funnels.
5. Backlink count and referring domain quality.
